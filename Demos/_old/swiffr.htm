Swiffr

view docs

version 1.0

Creates and returns a Flash object with version checking.

features

  • replaces the contents of a div with the embeded flash.
  • version checking.
  • add parameters to the flash object.
  • add properties to the flash object.
  • pass in variables to the flash movie.
  • built in javascript/actionscript communication gateway.

example

Flash player 8 or above is required to view this content.
