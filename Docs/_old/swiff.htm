Swiff

version 1.0

Creates and returns a Flash object with version checking.

syntax:

var mySwiff = new Swiff(path [, options]);

arguments:

  1. path - (string) - the location of the swf file.
  2. options - (object, optional)

options:

id
(string)
width
(number)
height
(number)
container
(element) the container into which the swf object will be injected.
params
(object) {allowScriptAccess:'always', quality:'high', swLiveConnect:true, wMode:'transparent'}
properties
(object) additional attributes for the object element.
vars
(object) vars will be passed to the swf as a querystring in flashvars.
version
(number) the min version number of the flash plugin. Default: 8

events:

Functions to call from the swf.
These will be available globally in the movie, and bound to the object.

returns:

  • (object) a new HTML object element.

view demoexample:

JavaScript


var obj = new Swiff('myMovie.swf', {
    id: 'myBeautifulMovie'
    width: 500,
    height: 400,
    params: {
        wmode: 'opaque',
        bgcolor: '#ff3300'
    },
    vars: {
        myVariable: myJsVar,
        myVariableString: 'hello'
    },
    events: {
        load: myOnloadFunc
    }
});

requirements:

  • mootools v1.2 core
