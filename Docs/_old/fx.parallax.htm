Fx.Parallax

version 1.0

add parallax control to the page.

syntax:

var para = new Fx.Parallax();

methods:

add

arguments:

  1. elements - (mixed, css selector or array)
  2. options - (object, optional)

options:

limit
(object) the amount in pixels of the maximum movement. (default: {w:100, h:100})
styles
(object) which styles should be used for the x & y movement. (default: {x: 'margin-left', y:'top'})
level
(number) which level of the parallax to add the elements too.
remove

arguments:

  1. el - (element) will remove the element in the specified level from the parallax.
  2. level - (number) specifies the level from which the element should be removed, if the element is null then the entire level is removed.

returns:

  • (object) a new Fx.Parallax instance.

view demoexample:

JavaScript


var para = new Fx.Parallax();
para.add([$('header')]);

requirements:

  • mootools v1.2 core
