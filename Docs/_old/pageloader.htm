pageLoader

version 1.1

loads content from pages via AJAX.

syntax:

var loader = new pageLoader([options]);

arguments:

  1. options - (object, optional)

options:

links
(mixed, array or CSS selector) the links that will get the content. Default: '.loadMe'
loadInTo
(mixed, element or string) the container that the content will be injected into. Default: content
loadFrom
(mixed, element or string) the container that the content will be taken from. Default: content

events:

_onStart
function to call when when the ajax request is made.
_onComplete
function to call when the content has been loaded.

returns:

  • (object) a new pageLoader instance.

view demoexample:

HTML


<a href="myExtraPage.htm" class="loadMe">Extra Page</a>

JavaScript


var loader = new pageLoader({
	links: $$('.loadMe')
});

requirements:

  • mootools v1.2 core
