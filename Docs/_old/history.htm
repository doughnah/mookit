history

version 1.0

enables browser back/forward buttons when using Javascript or AJAX.

syntax:

var myHistory = new history([options]);

arguments:

  1. options - (object, optional)

options:

links
(string) the css selector of the links that history should capture. Default: '.loadMe'

returns:

  • (object) a new history instance.

view demoexample:

JavaScript


var myHistory = new history({links: 'a.myLinks'});

requirements:

  • mootools v1.2 core
