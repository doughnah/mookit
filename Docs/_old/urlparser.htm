urlParser

version 1.0

splits any well-formed URL.

syntax:

parseUrl(url)

arguments:

  1. url - (string)

returns:

  • (object) containing: (source, protocol, authority, domain, port, path, directoryPath, fileName, query, anchor)

view demoexample:

JavaScript


var myUrl = parseUrl('http://platform.mandogroup.com/index.htm#javascript');
alert(myUrl.anchor);
