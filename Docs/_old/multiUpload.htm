multiUpload

version 1.0

Flash meets Ajax for simply working uploads. Based on FancyUpload by Harald Kirschner (digitarald)

syntax:

var uploader = new multiUpload(el, url [, options]);

arguments:

  1. el - (element)
  2. url - (string) - from the root of the site.
  3. options - (object, optional)

options:

limitSize
the max file size in Bytes. Default: false
limitFiles
the max number of files allowed to upload. Default: false
instantStart
start uploading files as soon as they are selected. Default: false
allowDuplicates
allow duplicate file to be uploaded. Default: false
fileTypes
what files are allowed to be uploaded.
path
the location of the Swiff.Uploader.swf. Default: /_common/swf/Swiff.Uploader.swf
validateFile
function to call that returns true or false.

events:

onLoad
function to call when the multiUpload instance has loaded.
fileInvalid
function to call when an invalid file is selected.
fileCreate
function to call to create a file element.
fileUpload
functio to call when the file is opened for upload.
fileComplete
function to call when the file has finished uploading.
fileRemove
function to call to remove the file from the queue.
OnAddFile
function to call when a file is added to the queue.

returns:

  • (object) a new multiUpload instance.

view demoexample:

JavaScript


var uploader = new multiUpload(
	$('uploadControls'),
	'/processUpload.aspx',
	{
		fileTypes: 'jpg,jpeg,gif,png',
		path: '/swf/Swiff.Uploader.swf'
	}
})

requirements:

  • mootools v1.2 core
  • Swiff.Uploader
