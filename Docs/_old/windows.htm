windows

version 1.0

draggable divs that snap to containers

syntax:

var win = new Windows(elements, containers [, options]);

arguments:

  1. elements - (array)
  2. containers - (array)
  3. options - (object, optional)

options:

handle
the classname of the item that is to be used as the window handle. (Default: handle)
containerBorder
number value of the border width of the containers. (Default: 0)

events:

OnDrop
the name of the function to execute on drop. The container is passed to the function if dropped in one.

returns:

  • (object) a new windows instance.
