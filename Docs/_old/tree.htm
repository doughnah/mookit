Tree

version 2.0

open & close nested lists.

syntax:

var tree = new Tree(el [, options]);

arguments:

  1. el - (mixed, element or string)
  2. options - (object, optional)

options:

openedCls
(string) the classname for the li when opened. Default: opened
closedCls
(string) the classname for the li when closed. Default: closed
trigger
(string) a css selector for the element inside the li that will open/close its nested list. Default: span
open
(string) the id of the li to open. Default: null

events:

_onClick
function to call when an item is clicked. The element & status is passed. (open/closed)

methods:

toggle

arguments:

  1. id - (string) the id of the li

returns:

  • (object) a new Tree instance.

view demoexample:

JavaScript


var myTree = new Tree($('myList'));

requirements:

  • mootools v1.2 core
