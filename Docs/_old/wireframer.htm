wireframer

version 0.1

draw boxes on a grid and move them around.

syntax:

var wire = new Wireframer(el [,options]);

arguments:

  1. el - (element)
  2. options - (object, optional)

options:

gridSize
the size of each grid element, specified in pixels. (default: 10)

events:

option
description

returns:

  • (object) a new wireframer instance.
