slideShow

version 1.4

transitions between images

syntax:

var show = new slideShow(container, images [, options]);

arguments:

  1. container - (mixed, element or string)
  2. images - (mixed, string or array) The string should be the classname on the a tags, the array should contain objects with the properties: path, title, description, url
  3. options - (object, optional)

options:

effect
(string) the type of transition. (fade|wipe|slide|random) Default: fade
duration
(number) the duration of the transition
transition
Default: Fx.Transitions.linear
direction
(string) the direction from which the next slide comes from. not used with the fade effect. (top|right|bottom|left|random) Default: right
wait
(number) the length of time to wait before showing the next slide.
loop
(boolean) show the first slide again when the last one is reached. Default: true
thumbnails
(mixed, array or string) show thumbnail indicator on these these thumbnails. Default: null
thumbnailCls
(string) the classname to be applied to the thumbnail to indicate that it is the current one. Default: outline
backgroundSlider
(object) a reference to a backgroundSlider instance. if used will overwrite the thumbnailCls. Default: null
loadingCls
(string) the classname for the loading gif. Default: loading

events:

onClick
function to call when a slide is clicked.

methods:

play
starts the slideshow.
stop
stops the slideshow.
next
goes to the next slide.
previous
goes to the previous slide.

returns:

  • (object) a new slideShow instance.

view demoexample:

JavaScript


var show = new slideShow(
	$('container'),
	'slideShowImages',
	{
		wait: 3000,
		duration: 1000,
		effect: 'fade'
	}
);

requirements:

  • mootools v1.2 core
