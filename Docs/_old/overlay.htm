overlay

version 1.2

covers the window with a semi-transparent layer.

syntax:

var myOverlay = new overlay([options]);

arguments:

  1. options - (object, optional)

options:

colour
(string) the color of the layer. Default: #000
opacity
(number) the value the layer should fade too. (0-1) Default: 0.7
zIndex
(number) the z-index of the layer. Default: 1
container
(element) the element that the layer should cover. Default: document.body

events:

_onClick
function to call when the layer is clicked.

methods:

show
 
hide
 

returns:

  • (object) a new overlay instance.

view demoexample:

JavaScript


var myOverlay = new overlay({
	onClick: function(){
		this.hide();
	}
});
myOverlay.show();

requirements:

  • mootools v1.2 core
