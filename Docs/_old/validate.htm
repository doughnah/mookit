validate

version 2.2

form validation.

syntax:

var val = new validate(el [, options]);

arguments:

  1. el - (mixed, element or string)
  2. options - (object, optional)

options:

validateOnBlur
(boolean) validates the input on the onBlur event. Default: true
errorClass
(string) the classname to be applied to the input on error. Default: error
errorMsgClass
(string) the classname to be applied to the error message. Default: errorMessage
dateFormat
(string) the format which dates must be specified in. Default: dd/mm/yy
showErrorsInline
(boolean) show the error message next to the input. Default: true
label
(string) the text to be displayed on the submit button once clicked. Default: Please wait...

events:

_onSuccess
function to call when the form has passed validation on submit. Passes a reference to the form.
_onFail
function to call when the form fails validation on submit. Passes an array of errors.

returns:

  • (object) a new validate instance.

view demoexample:

HTML


<label for='name'>name:</label> <input type='text' name='name' class='required'>
<label for='email'>email:</label> <input type='text' name='email' class='required email'>

JavaScript


var val = new validate($('myForm'));

requirements:
