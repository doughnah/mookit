Fx.Bezier

version 1.0

bezier transition

Inherits methods, properties, options and events from Fx.Morph

syntax:

var myFx = new Fx.Bezier(element [, options]);

arguments:

  1. element - (mixed) A string ID of the Element or an Element to apply the style transitions to.
  2. options - (object, optional)

options:

styles
(object) the styles which represent the x and y movement. Default: {x:'left', y:'top'}
controls
(array) upto 3 objects specify the x & y control points for the curve. (e.g. [{x:100, y:100}])

plus Fx.Morph options

events:

see Fx.Morph events

returns:

  • (object) a new Fx.Bezier instance.

view demoexample:

JavaScript


var myFx = new Fx.Bezier($('myDiv'), {
	duration: 1500, 
	transition: Fx.Transitions.Sine.easeInOut, 
	controls: [{x:360,y:-300},{x:360,y:600}]
})

myFx.start({top: [100,400], left: [100,400]});
