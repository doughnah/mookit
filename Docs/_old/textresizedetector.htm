textResizeDetector

version 1.0

captures when the text is resized.

syntax:

var detector = new textResizeDetector([options]);

arguments:

  1. options - (object, optional)

options:

delay
(number) the amount of time (ms) to pause between checking. Default: 200

events:

_onChange
function to call when the text has been resized.

returns:

  • (object) a new textResizeDetector instance.

view demoexample:

JavaScript


var detector = new textResizeDetector({
	onChange: function(){
		alert('text changed');
	}
});

requirements:

  • mootools v1.2 core
