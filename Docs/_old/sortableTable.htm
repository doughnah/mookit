sortableTable

version 1.4

column sorting and row filtering on tables

syntax:

var table = new sortableTable(el [, options]);

arguments:

  1. el - (mixed, element or string)
  2. options - (object, optional)

options:

overCls
(string) the classname for the row when the mouse is over it. Default: false
sortOn
(number) which column to initially sort on (zero-based). Default: 0
sortBy
(string) which direction to initially sort on (ASC / DESC). Default: ASC
filter
(boolean) adds the filtering options above the table. Default: true
filterHide
(boolean) when filtering, show only the matches, hide the rest. Default: true
height
(number) fixed the height of the table, overflow will be scrolled. Default: false
linkId
(number) the column number in which the a href link is (zero-based). Default: 0

events:

onClick
function to call when the row is clicked. Passed: url

returns:

  • (object) a new sortableTable instance.

view demoexample:

JavaScript


var table = new sortableTable($('myTable'), {
	overCls: 'overRow',
	sortOn: 1, // 2nd column (date)
	sortBy: 'DESC', // latest first
	onClick: function(url){
		alert(this.id);
		alert(url);
	}
})

requirements:
