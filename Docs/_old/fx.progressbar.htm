Fx.ProgressBar

version 1.0

an animated progrees bar indicator

syntax:

var myFx = new Fx.ProgressBar((el [, options]);

arguments:

  1. el - (element)
  2. options - (object, optional)

options:

text
(element) to update with percentage text
transition
default: Fx.Transitions.Circ.easeOut

returns:

  • (object) a new Fx.ProgressBar instance.

view demoexample:

JavaScript


var myProgress = new Fx.ProgressBar($('progress'), {
	text: new Element('span').inject($('progress'), 'after')
});

requirements:

  • mootools v1.2 core
