backgroundSlider

version 1.3

slides a layer to a given elements position and dimensions.

syntax:

var bgSlider = new backgroundSlider(elements [, options]);

arguments:

  1. elements - (array)
  2. options - (object, optional)

options:

duration
(number) the duration of the transition. Default: 300
wait
(number) the amount of time before the slider should reset. Default: 500
transition
Fx.Transitions.Sine.easeInOut
id
(mixed, element or string) the div to use as the slider, if null a div will be created.
className
(string) the classname to be applied to the slider div if created (id=null)
start
(number) the number of the element to have the slider as default, otherwise the first element is default. (1)
fixHeight
(number) a number to fix the height of the slider. It will not use the target's properties. Default: null
fixWidth
(number) a number to fix the width of the slider. It will not use the target's properties. Default: null
padding
(object) top, right, bottom, left values to add padding around the target element.
mouseOver
(boolean) the slider will move to the target elements position on mouseover. Default: true

events:

onClick
function to call when the target element is clicked. A reference to the element of passed in.

returns:

  • (object) a new backgroundSlider instance.

view demoexample:

JavaScript


var bgSlider = new backgroundSlider($$('myMenu a'));

requirements:

  • mootools v1.2 core
